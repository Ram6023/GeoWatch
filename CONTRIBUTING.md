# Contributing to GeoWatch

Thank you for your interest in contributing to GeoWatch!

## How to Contribute

1. **Fork the Repository**: Create your own fork of the project.
2. **Clone the Project**: Clone your fork to your local machine.
3. **Create a Feature Branch**: `git checkout -b feature/your-feature-name`
4. **Commit Your Changes**: `git commit -m 'Add some feature'`
5. **Push to the Branch**: `git push origin feature/your-feature-name`
6. **Open a Pull Request**: Submit a pull request to the main repository.
